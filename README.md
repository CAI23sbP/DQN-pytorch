# DQN-pytorch
Deep Q-value Network by pytorch

## install ##
`git clone https://github.com/CAI23sbP/DQN-pytorch.git`

`pip install -r requirements.txt`

## Training  ##
`python3 train.py`

## Testing  ##
`python3 test.py`

## See detail config ## 
open `configs.py`
